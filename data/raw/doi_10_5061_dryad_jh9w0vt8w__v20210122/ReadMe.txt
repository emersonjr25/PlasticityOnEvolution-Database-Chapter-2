ReadMe.txt - This summary file.

PHYLOGRAM ESTIMATE
turtle.nex - full data matrix of 593 individuals.
turtle.con.tre - majority rule consensus phylogram of full data matrix. Depicted in Fig. S1A-E.
turtle.mcc.tre - maximum clade credibility tree of full data matrix.

CHRONOGRAM ESTIMATE
turtle.288.bd.xml - reduced dataset, retaining one individual per species, used in the divergence time analysis.
bd.mcc.median_heights.tre - Maximum clade credibility tree arising from the divergence time analysis. Depicted in Figure 1A and B.
sampleFromPosterior.100.tre - Sample of 100 posterior trees from the divergence time analysis that TESS and GeoHiSSE analyses were iterated over to check for sensitivity to the tree.

TREE WIDE DIVERSIFICATION SHIFT ANALYSIS
turtles_CoMET.R - R script for TESS and COMET analyses.
TESSInput.tre - Input tree for TESS analyses.

GEOGRAPHY DEPENDENT DIVERSIFICATION ANALYSIS
GeoHiSSE.final_analyses.R - R script that uses HiSSE and dependencies to fit diversification models, reconstruct ranges, and average results.
GeoHisseInput.tree - Input tree for GeoHiSSE.
GeoHiSSEInput.turtles_coast1_both0_not2.final.nex - Input range data for GeoHiSSE.
